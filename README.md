# 微博抽奖工具

免费、公平、可验证的微博转发抽奖工具。无需 SSVIP，任何人都可以使用。

🚀 **在线使用**：[weibo-lottery.pages.dev](https://weibo-lottery.pages.dev)

---

## ✨ 特性

- **公平抽奖** — 基于 SHA-256 密码学哈希的随机算法，过程透明可复验
- **公开验证** — 任何人可用相同种子+名单复算结果，确保未被篡改
- **三种种子来源** — 中国福彩开奖号码（默认）、比特币区块哈希、自定义
- **名单承诺** — 开奖前公布名单哈希，防止开奖后偷换名单
- **浏览器端抓取** — 在你自己的浏览器里运行脚本抓取转发用户，不经过服务器，绕开微博 IP 风控
- **分享公示** — 一键生成分享链接，他人打开即可查看结果并验证

## 🔧 使用方法

### 1️⃣ 打开微博帖子
在浏览器登录微博，进入要抽奖的那条微博的转发列表页面（推荐手机版 `https://m.weibo.cn/status/帖子ID`）。

### 2️⃣ 运行抓取脚本
复制工具页面提供的脚本 → 在微博页面按 `F12` → Console → 首次粘贴需先输入 `allow pasting` 回车 → 粘贴脚本运行。脚本会自动翻页抓取全部转发用户，完成后把名单复制到剪贴板。

> **为什么在浏览器里跑，而不是填 Cookie 让服务器抓？**
> 微博对登录态有 IP 风控：Cookie 在国内登录，但云端服务器是海外 IP，会被判定为异地风险、强制重新登录（错误码 -100）。让脚本在你自己的浏览器里运行，用的是你本人的 IP 和登录态，不触发风控，成功率最高，也无需把 Cookie 交给任何服务器。

### 3️⃣ 粘贴名单并抽奖
回到工具，把名单粘贴进去 → 设置中奖人数 → 选择种子来源（推荐福彩开奖号码）→ 开始抽奖。

### 4️⃣ 公示与验证
生成分享链接，他人打开即可查看完整结果；验证页支持一键复算，确认结果未被篡改。

## 🎲 抽奖算法

```
对每位参与者计算: SHA-256("种子:用户名:名单哈希")
按哈希值升序排列，前 N 名即为中奖者
```

- **种子**：公开可查的随机源（福彩号码/比特币区块哈希）
- **名单哈希**：所有参与者排序后的 SHA-256 摘要，开奖前公布
- 任何人拿到种子和名单，都可以独立复算出相同结果
