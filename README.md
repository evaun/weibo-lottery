# 微博抽奖工具

免费、公平、可验证的微博转发抽奖工具。无需 SSVIP，任何人都可以使用。

🚀 **在线使用**：[weibo-lottery.pages.dev](https://weibo-lottery.pages.dev)

---

## ✨ 特性

- **公平抽奖** — 基于 SHA-256 密码学哈希的随机算法，过程透明可复验
- **公开验证** — 任何人可用相同种子+名单复算结果，确保未被篡改
- **三种种子来源** — 中国福彩开奖号码（默认）、比特币区块哈希、自定义
- **名单承诺** — 开奖前公布名单哈希，防止开奖后偷换名单
- **云端抓取** — 无需安装 Python，网页端直接抓取转发用户
- **分享公示** — 一键生成分享链接，他人打开即可查看结果并验证

## 🔧 使用方法

### 1️⃣ 获取 Cookie

1. 打开 [m.weibo.cn](https://m.weibo.cn) 并登录微博
2. 按 `F12` 打开开发者工具，切到 Console 标签
3. 首次粘贴代码时，Chrome 会提示安全警告，先输入 `allow pasting` 并回车
4. 复制工具页面中的代码，粘贴到 Console 运行，Cookie 自动复制

### 2️⃣ 抓取转发用户

- 在工具中粘贴 Cookie 和微博帖子 ID（支持短 ID 如 `R1vno65Lz`）
- 点击「开始抓取」，自动获取所有转发用户
- Cookie 仅用于本次请求，不会被存储

### 3️⃣ 开始抽奖

- 设置中奖人数
- 选择种子来源（推荐使用福彩开奖号码）
- 点击「开始抽奖」

### 4️⃣ 公示与验证

- 生成分享链接，他人打开即可查看完整结果
- 验证页支持一键复算，确认结果未被篡改

## 🎲 抽奖算法

```
对每位参与者计算: SHA-256("种子:用户名:名单哈希")
按哈希值升序排列，前 N 名即为中奖者
```

- **种子**：公开可查的随机源（福彩号码/比特币区块哈希）
- **名单哈希**：所有参与者排序后的 SHA-256 摘要，开奖前公布
- 任何人拿到种子和名单，都可以独立复算出相同结果

## 🐍 Python 离线脚本（备用）

```bash
python3 fetch_reposts.py <帖子ID> [Cookie]
```

- 交互式输入 Cookie，自动缓存到本地
- 支持短 ID 自动转换
- 输出参与者名单到文本文件
