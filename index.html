使用方法（只需两步）

第一步：获取 Cookie
① 用 Chrome 打开 m.weibo.cn 并登录微博
② 按 F12 打开开发者工具 → 点顶部「Network」
③ 在微博页面上随便点一下或刷新
④ 左侧点出现的请求 → 右侧找 Request Headers → Cookie
⑤ 双击 Cookie 值全选，Ctrl+C 复制

第二步：运行脚本
① 终端运行 python3 fetch_reposts.py 微博ID
② 粘贴刚才复制的 Cookie，按回车
③ 脚本自动抓取，生成 reposts_微博ID_names.txt
④ 回到这里点上方「导入文件」上传该 TXT

💡 Cookie 会自动缓存到本地，下次不用再输入

验证方法

如何验证：
1. 获取参与者名单，按 Unicode 码点升序排列
2. 计算 SHA-256(名单.join(",")) 得到名单哈希
3. 对每个参与者，计算 SHA-256(种子 + ":" + 用户名 + ":" + 名单哈希) 得到分数
4. 按分数从小到大排序，前 N 名即为中奖者
5. 与公示结果比对，完全一致则验证通过

或者：点击下方"生成独立验证页"，得到一个自包含 HTML 文件，浏览器打开即可一键验证。
