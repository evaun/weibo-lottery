在网页上直接抓取微博转发用户，无需安装任何软件。只需粘贴你的微博 Cookie 和帖子 ID 即可。

方式 A（推荐）：复制下方代码 → 打开 m.weibo.cn → F12 → Console → 粘贴运行

⚠️ 首次粘贴时，Chrome 会提示 Warning: Don't paste code…，这是安全提示。请先在控制台输入 allow pasting 并按回车，然后才能粘贴代码。
void(function(){if(location.hostname!=='m.weibo.cn'){alert('请先打开 m.weibo.cn 并登录微博');return}var c=document.cookie;if(!c){alert('未找到Cookie，请确保已登录微博');return}var t=document.createElement('textarea');t.value=c;t.style.cssText='position:fixed;left:-9999px';document.body.appendChild(t);t.select();try{document.execCommand('copy');var d=document.createElement('div');d.textContent='✅ Cookie已复制！回到抽奖工具粘贴即可';d.style.cssText='position:fixed;top:20px;left:50%;transform:translateX(-50%);background:#4ade80;color:#000;padding:14px 28px;border-radius:10px;font-size:15px;font-weight:700;z-index:99999;box-shadow:0 4px 20px rgba(0,0,0,0.3)';document.body.appendChild(d);setTimeout(function(){d.remove()},3000)}catch(e){prompt('自动复制失败，请手动复制下方Cookie：',c)}t.remove()}).call(this)

验证方法

如何验证：
1. 获取参与者名单，按 Unicode 码点升序排列
2. 计算 SHA-256(名单.join(",")) 得到名单哈希
3. 对每个参与者，计算 SHA-256(种子 + ":" + 用户名 + ":" + 名单哈希) 得到分数
4. 按分数从小到大排序，前 N 名即为中奖者
5. 与公示结果比对，完全一致则验证通过
